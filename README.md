# Python Job Scraper

This project is a simple Python web scraper that collects job listings from a sample job website.

## 🌐 Project URL
https://realpython.github.io/fake-jobs/

## 📌 Features
- Scrapes job listings from the website
- Extracts:
  - Job Title
  - Company Name
  - Location
  - Job Link
- Saves data into a CSV file

## 🛠️ Technologies Used
- Python
- requests
- BeautifulSoup
- pandas
